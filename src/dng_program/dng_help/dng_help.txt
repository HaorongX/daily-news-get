command
-v, -V                           Print Version infomation
-h, -H                           Print Help infomation
install                          Install script to dng
remove, uninstall        Uninstall script from dng
list                               List the scripts installed
open                            Set scripts status to 'open'
close                           Set scripts status to 'close'
execute                       Execute the scripts 
change, update           Change the scripts installed 
setoutput                     Set the result output destination(The default destination is your screen)
